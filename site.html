How to Set Up an SSH Server using Ubuntu Server

by Mikołaj Sołtysiński (257202)

Step 1: Download Ubuntu Server ISO

Download the Ubuntu Server ISO from the official website, load it into your virtual machine, and start the machine.

Step 2: Install SSH Server Package During Installation

Proceed through the Ubuntu installation process and choose the option to install the openssh-server.

Step 3: Verify if SSH is Running

Check if the SSH service is running:

sudo systemctl status ssh

If it is not running, start it with:

sudo systemctl enable ssh && sudo systemctl start ssh

Step 4: Edit SSH Configuration

Edit the SSH configuration file to make necessary changes:

sudo nano /etc/ssh/sshd_config

While you are there take a look at the port used by SSH (Default is 22)

Save the file and exit the editor after making your changes.

Step 5: Check IP Address

Find the server's IP address to use for SSH connections:

hostname -I

Note the IP address for later use when connecting from another machine.

Step 6: Add a Guest Account

Create a new guest account and set a password:

sudo adduser guest
sudo passwd guest

Type the new password twice.

Step 7: Set Up SSH Client on Another Machine

Ensure the SSH client package is installed on another Linux virtual machine:

sudo apt install openssh-client

Step 8: Connect to the SSH Server

Use the following command to connect to the remote server:

ssh <username>@<server-ip>

If you have set a different port use:

ssh <username>@<server-ip> -p <port>

Step 9 (Optional): Use PuTTY to connect from windows

Download the PuTTY software from the official site.

Run it and in the Host Name put:

<username>@<server-ip>

Then click on open to connect to your server.
